# File Tree

```
deepl-xml-translator/
├── src/                                  # All library source code (DeeplXML namespace)
│   ├── Translator.php                    # Main entry point: the Translator class
│   └── Translator/
│       ├── Exception.php                 # Base exception for the library
│       ├── String.php                    # Container for a single translatable string
│       └── Exception/
│           └── Request.php               # Specialised exception for failed DeepL requests
│
├── tests/                                # PHPUnit test suite
│   ├── bootstrap.php                     # Test bootstrapper; loads apikey.php if present
│   ├── apikey.dist.php                   # Template for API key / proxy config (commit safe)
│   ├── apikey.php                        # Local API key config (gitignored, not committed)
│   └── testsuites/
│       └── testsuites/
│           └── TranslatorTests.php       # All unit and live-API integration tests
│
├── docs/                                 # Documentation and agent manifests
│   ├── config/                           # (existing docs sub-directory, not analysed)
│   ├── phpstan/                          # PHPStan-related docs
│   └── agents/
│       └── project-manifest/             # This manifest (generated by Manifest Curator)
│
├── vendor/                               # Composer-managed dependencies (not committed)
├── composer.json                         # Package metadata, dependencies, scripts
├── composer.lock                         # Locked dependency versions
├── composer.phar                         # Committed Composer PHAR for self-contained use
├── phpunit.xml                           # PHPUnit configuration
├── phpstan.neon                          # PHPStan configuration (level 6, src/ only)
├── run-tests.bat                         # Windows batch helper to run tests
├── changelog.md                          # Human-readable version history
├── LICENSE                               # GPL-3.0-or-later
└── README.md                             # Project overview and usage guide
```

## Notes

- `vendor/` is excluded from static analysis (see `phpstan.neon`).
- `tests/` is also excluded from static analysis.
- `tests/apikey.php` is gitignored (not checked in). `tests/apikey.dist.php` is the template.
- The nested `tests/testsuites/testsuites/` path is the actual location PHPUnit scans (configured in `phpunit.xml`).
